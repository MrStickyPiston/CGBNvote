Install python
Run setup.py
Run 'server.py' as admin
Go to 'http://localhost:80/vote-admin', login and scroll down to voting active. Change 0 to zero to enable.
You can also edit the candidates there. At default only a few examples are listed.
If you want to have everyone access to it, google 'port forwarding tutorial'
Check what your ip is. People will need this to vote.
Tell everyone to go vote on http(s)://[YOUR IP|DOMAIN]/vote.
